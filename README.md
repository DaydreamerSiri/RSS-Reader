# RSS-Reader

Unterstützte Plattformen : Windows, MacOS, Linux

Ein RSS Reader, welches Microsoft RSS Feed abließt.

Double Click Eintrag oder Eintrag auswählen und auf öffnen Link klicken damit der Eintrag geöffnet wird in einem Browser.

Filter Textbox kann genutzt werden um Filter nach Einträgen zu suchen (Achte auf Rechtschreibung)

Das klicken auf die Column Headers von den jeweiligen Columns sortiert die Einträge Auf-/ Absteigend nach den jeweiligen Datentypen (Artikel nach Alphabet, Datum nach derjeweiligen Veröffentlichungsdatum).
